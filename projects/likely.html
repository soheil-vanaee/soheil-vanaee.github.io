process

UX approach

I designed for a phone passed between people in a group, not one person scrolling alone, which changed almost every UI decision.

  • Mapped the full game loop: setup, prompts, answers, voting, results
  • Wireframed flows optimized for quick handoffs between players
  • Designed high‑fidelity screens with playful, legible‑at‑a‑glance UI
  • Prototyped voting and results screens for in‑person usability testing
  • Iterated on pacing so rounds felt fast, not rushed
